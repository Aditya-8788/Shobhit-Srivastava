# Task 4: Food Hunt Adventure

## Overview

Food Hunt Adventure is a Flutter mobile learning app for children aged 3–8.  
The child uses the phone camera to capture food items around them. The app detects the captured item using image labeling and rewards the child with stars when food is found.

## Features

- Camera integration
- Capture image feature
- Food identification using Google ML Kit
- Star reward system
- Progress tracking: 5 food items
- Completion reward screen
- Confetti animation
- Child-friendly UI
- Clean Architecture
- BLoC state management

## Tech Stack

- Flutter
- Dart
- BLoC
- Clean Architecture
- Camera API
- Google ML Kit Image Labeling
