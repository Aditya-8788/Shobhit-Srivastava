# Task 2: Balloon Pop Adventure

## Overview

Balloon Pop Adventure is a single-screen interactive Flutter game designed for children aged 3–5 years. The child taps colorful floating balloons, hears a pop sound, and earns points. After all balloons are popped, the child can play again with balloons appearing in new random positions.

## Features

- Child-friendly colorful UI
- Tap interaction
- Balloon pop sound
- Floating balloon animation
- Score counter
- Random balloon positions
- Non-overlapping balloons
- Play Again option
- Clean Architecture
- BLoC state management

- Flutter
